## PYTHON3  
Создаем файл *hello.py* в `C:\cygwin\home\<user>`

```python
#!/usr/bin/env python3
print("Hello, World!")
```

Далее для теста в *cygwin* пишем:
```sh
$ python3 -V
$ chmod a+x hello.py
$ ./hello.py
Hello, World!
```

Устанавливаем библиотеку через [**pip3**](pip3.md):  
```sh
$ pip install requests
```
